# Heartbeat


## Setup

####Dependencies

- Node (http://nodejs.org/)
- Express (http://expressjs.com/)
- Socket.io (http://socket.io/)

#### Dev Dependencies

- node supervisor (https://github.com/isaacs/node-supervisor)
- bower.io (http://bower.io/)


    npm install


## Run

    node index.js

or fancy start

    npm start

## Watch and restart.

    supervisor index.js
